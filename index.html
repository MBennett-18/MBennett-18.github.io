Today's Weather and Colors

This web applet uses the weather description and hands it to the Colr API to find colors tagged with the description. This random color is passed to TheColorAPI to generate a pallete using the random seed color. This provides a color pallete that tries to capture the current weather conditions.

Flu Twitter

Work in Progress

Using the Twitter API, tweets containing the words "Flu" or "Influenza" around the Maritimes are collected and analyzed. Tweets are gathered daily via a Python script and the resulting JSON is loaded. Currently undergoing frequent edits.
